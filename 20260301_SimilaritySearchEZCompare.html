Descriptor Comparison — 3×3 Similarity Search

This tool runs three simultaneous Tanimoto similarity searches using three independently configured molecular fingerprints and displays the top-3 hits for each descriptor in a 3×3 matrix.

How to use

1. Enter a query SMILES in the left panel.
2. Paste or load a library of SMILES.
3. Configure each of the three descriptor columns (fingerprint type and parameters).
4. Click Compare All Descriptors.

Fingerprints available

Morgan (ECFP) — circular fingerprints. Radius 2 = ECFP4.
RDKit Topological — path-based fingerprints.
MACCS Keys — 166-bit structural keys (fixed).
Atom Pair — atom-pair count fingerprints.
Topological Torsion — 4-atom torsion angle fingerprints.

Overlap analysis

Below the matrix, unique hits, pairwise overlaps, and triple overlaps across the three descriptor rankings are highlighted — showing which compounds rank consistently across different chemical representations.
