About Compound Similarity Search

What is this tool?

This web application performs molecular similarity searches using various fingerprint methods. It compares a query molecule against a library of compounds to find the most similar structures based on Tanimoto similarity. Author: Vladimir Chupakhin, Data sources (compounds): wikipedia + AI-based SMILES annotation (curation steps were minimal so mistakes might be still present).

Fingerprint Types

  • Morgan (ECFP): Circular fingerprints based on atom neighborhoods. Most commonly used for general similarity searching.
  • RDKit Topological: Path-based fingerprints encoding molecular graph topology.
  • MACCS Keys: 166 predefined structural keys. Fast and interpretable.
  • Atom Pair: Encodes atom pairs and their topological distances.
  • Topological Torsion: Encodes sequences of 4 connected atoms.

Technology

Built with RDKit.js, the JavaScript port of the RDKit cheminformatics toolkit. All calculations run entirely in your browser - no data is sent to any server.

Usage Tips

  • Start with Morgan fingerprints (radius=2) for general similarity
  • Larger nBits values reduce collisions but increase computation time
  • Try different fingerprint types for different perspectives on similarity
  • MACCS Keys are fastest for large libraries
